## 概念
在空对象模式(`Null Object Pattern`)中，一个空对象取代 `NULL` 对象实例的检查。`Null` 对象不是检查空值，而是反应一个**不做任何动作**的关系。这样的 `Null` 对象也可以在**数据不可用**的时候提供**默认**的行为。
在空对象模式中，我们创建一个指定各种要**执行的操作的抽象类和扩展该类**的实体类，还创建一个**未对该类做任何实现的空对象类**，该空对象类将无缝地使用在需要检查空值的地方
