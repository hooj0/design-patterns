## 概念
观察者模式(`Observer Pattern`)：定义对象间的**一种一对多**依赖关系，使得每当一个对象**状态发生改变**时，其**相关依赖对象皆得到通知并被自动更新**。观察者模式又叫做**发布-订阅**（`Publish/Subscribe`）模式、**模型-视图**（`Model/View`）模式、**源-监听器**（`Source/Listener`）模式或**从属者**（`Dependents`）模式。观察者模式是一种对象**行为型模式**。
