## 概念
组合模式（`Composite Pattern`），又叫部分整体模式，是用于把一组**相似的对象当作一个单一的对象**。组合模式依据**树形结构**来组合对象，用来表示部分以及整体层次。这种类型的设计模式属于结构型模式，它创建了**对象组的树形结构**。
