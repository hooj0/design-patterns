## 概念
享元模式(`Flyweight` )：运用**共享技术**有效地支持**大量细粒度**对象的复用。系统只使用**少量的对象**，而这些对象都**很相似，状态变化很小**，可以**实现对象的多次复用**。由于享元模式要求能够共享的对象必须是细粒度对象，因此它又称为**轻量级模式**，它是一种对象结构型模式。简单来说：它用于通过尽可能多地与类似对象**共享来最小化内存**使用或计算开销。
